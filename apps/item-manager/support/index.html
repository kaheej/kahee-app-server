사용 가이드

시작하기

Item Manager는 개인 소유물을 체계적으로 관리할 수 있는 앱입니다. 처음 시작할 때 다음 단계를 따라해보세요:

  1. 앱을 실행하고 첫 아이템을 추가해보세요
  2. 카테고리를 선택하여 물품을 분류하세요
  3. 사진을 추가하여 시각적으로 관리하세요
  4. 보관 위치를 상세히 설정하여 물건을 쉽게 찾으세요 (예: 거실 서랍 2번째 칸)

주요 기능

  • 물품 등록: 구매일, 가격, 보증기간 등 상세 정보 입력
  • 사진 관리: 물품당 여러 장의 사진 첨부 가능
  • 카테고리 분류: 20개의 카테고리로 체계적인 분류
  • 검색 기능: 이름, 카테고리, 위치 등으로 빠른 검색
  • 통계 보기: 보유 물품 통계 및 지출 분석
  • 보관 장소 관리: 물건의 위치를 정확하게 기록하고 관리

자주 묻는 질문

데이터는 어디에 저장되나요? +
모든 데이터는 사용자의 기기에 로컬로 저장됩니다. 클라우드 서버를 사용하지 않으므로 인터넷 연결 없이도 사용 가능합니다.
보관 장소는 어떻게 관리하나요? +
물품 등록 시 보관 장소를 입력하고, 위치별로 분류하여 관리할 수 있습니다. 검색 기능을 통해 특정 위치에 있는 물건들을 쉽게 찾을 수 있습니다.
사진 용량 제한이 있나요? +
앱 자체에는 제한이 없지만, 기기의 저장 공간에 따라 달라집니다. 고화질 사진은 자동으로 압축되어 저장됩니다.
